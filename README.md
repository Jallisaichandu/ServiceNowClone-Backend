# ServiceNow Clone Backend

A Spring Boot REST API for an IT Ticket Management System.

## Features

- JWT Authentication
- Spring Security
- Role-Based Authorization
- Ticket CRUD
- Comments
- Notifications
- Excel Report Generation
- MySQL Database

## Technologies

- Java 21
- Spring Boot
- Spring Security
- JWT
- Hibernate
- MySQL
- Maven

## Deployment

Railway
